# PySLM
Pythonic Sound Level Meter.


## Usage
To begin, one can try:

    >>> import pyslm
    >>> pyslm.run()


## Installation

    >>> pip install git+https://github.com/pyttamaster/pytta@development


## Dependencies
- Numpy;
- Scipy;
- Matplotlib;
- Sounddevice;
- PySide2;
- H5py;
- pyqtgraph.
